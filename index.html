About Me

백엔드를 넘어 인프라로 확장하는 엔지니어

백엔드 개발자로 시작해 Spring Boot 기반의 API 개발과 아키텍처 설계를 경험하며 서비스의 확장성과 안정성의 중요성을 배웠습니다. 현재는 이 지식을 인프라 운영 영역으로 확장하여, 데브옵스(DevOps)와 클라우드 엔지니어링을 집중적으로 학습하고 있습니다. 단순히 코드를 작성하는 것을 넘어, AWS와 Kubernetes를 활용한 컨테이너 오케스트레이션, CI/CD 파이프라인 구축을 통해 코드가 실제 서비스로 배포되고 운영되는 전체 과정을 자동화하는 데 집중합니다. 기존에 쌓아온 코드 리뷰와 명확한 커뮤니케이션 능력을 바탕으로, 개발과 운영 환경 전반의 효율을 높이는 엔지니어가 되겠습니다.
